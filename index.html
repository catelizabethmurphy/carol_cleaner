Getting CAROL data
How to download your files
Summary CSV — required
  1. Open the new CAROL site.
  2. Set the record type to Safety Recommendations and build your query (filter by mode, date, recipient, and so on).
  3. Run the query, then use the Export / Download option and choose CSV.
  4. Drop the file into the box, or use Choose files.
Full JSON — optional
  1. Request the full recommendation records from the CAROL API as a JSON array.
  2. Keep each record's srid field — that's how rows are matched to the CSV.
  3. Save it as a .json file and drop it into the same box.
